# Screenshot Placeholders

Currently, this folder is ready for screenshots. 

To populate with actual screenshots:
1. Run the app in Xcode Simulator
2. Navigate to each screen
3. Press ⌘S to take screenshots
4. Move screenshots here and rename them

Expected files:
- patients_list.png
- patient_detail.png
- add_patient.png
- schedule.png
- add_appointment.png
- ai_screen.png
- settings.png
